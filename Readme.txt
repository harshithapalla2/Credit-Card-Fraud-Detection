Steps to run the code

1. Open .ipynb in your IDE or Jupyter Lab 
2. Run all cells in the .ipynb file (make sure you run the pickle dump code)
3. Run !pip install flask
4. Run pip install flask_cors
5. Running flask api cell will start the flask server on http://127.0.0.1:5000 
6. Open home.html file(in the templates folder) with a local browser
7. Play around with the website!
